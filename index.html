Fundamentación Teórica

Internet:Internet es una red de computadoras que se encuentran interconectadas a nivel mundial para compartir información. Se trata de una red de equipos de cálculo que se relacionan entre sí a través de la utilización de un lenguaje universal. (Etecé, 2021)

La Web:Es un sistema interconectado de páginas web públicas accesibles a través de Internet. La Web no es lo mismo que el Internet: la Web es una de las muchas aplicaciones construidas sobre Internet. (Significados.com, s.f.)

El diseño web:El diseño web es un área enfocada en el desarrollo de interfaces digitales, como el diseño de sitios y aplicaciones para web. Para ello, los diseñadores web crean las páginas utilizando lenguajes de marcado como HTML. (Rockcontent, 2019)

Pagina Web:Es un documento digital complejo, que puede integrar y/o contener texto, sonio, video, programas, enlaces, imágenes, hipervínculos y otros elementos, adaptado para la WWW, y que puede ser accedida y visualizada mediante un navegador WEB. Esta información se encuentra generalmente en etiquetado HTML o XHTML, y puede proporcionar acceso a otras páginas WEB mediante enlaces de hipertexto. Frecuentemente también incluyen otros recursos como pueden ser hojas de estilo, scripts, imágenes digitales, entre otras. (Wikipedia, 2023)

Lenguaje HTML,HTML:siglas en inglés de HyperText Markup Language (‘lenguaje de marcado de hipertexto’), hace referencia al lenguaje de marcado para la elaboración de páginas web. Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros. (Wikipedia, 2023)
